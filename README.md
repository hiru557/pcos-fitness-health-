# PCOS Fitness & Health Website

This is a **PCOS (Polycystic Ovary Syndrome) Fitness & Health** website designed to help users manage their diet, workouts, and track progress. It also includes a chatbot for queries related to PCOS.

## Features
- **Home Page** (`index.html`)  
- **Diet Plans** (`Diet.html`)  
- **Workout Plans** (`Workout.html`)  
- **Progress Tracking** (`Progress.html`)  
- **Chatbot** (`chatbot.html`)  
- Responsive design using `style.css`  
- Interactive functionality with `app.js`
